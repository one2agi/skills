Core Features

Three Pillars of Improvement

🚀

Real-Time Capture

Hook intercepts messages at preprocessed stage, detecting corrections, errors, and feature requests in real-time with multilingual support.

Node.js Hooks keyword detection session routing
⚙

Pattern Detection

Hourly cron runs distill script to scan learnings, aggregate by Pattern-Key, and identify recurring mistakes (count ≥ 2 threshold).

Bash Scripts AWK aggregation threshold logic
🔔

Smart Notifications

Channel-specific bots deliver actionable notifications. User responds with A/B/C/D, triggering the correct agent session to execute improvements.

per-agent routing Telegram/Discord A/B/C/D decisions
